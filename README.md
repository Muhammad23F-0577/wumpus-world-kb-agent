# Wumpus World KB Agent

An AI-powered Wumpus World simulator built with React.js using Knowledge Base reasoning, CNF conversion, and Resolution Algorithm to detect safe paths, avoid pits/Wumpus, and find gold.

## Features

- Dynamic grid world generation
- Random pits, Wumpus, and gold placement
- Breeze and stench percept detection
- Knowledge Base (KB) reasoning
- CNF clause conversion
- Resolution-based safe path finding
- Interactive UI with live agent movement

## Requirements

- Node.js installed
- npm installed
